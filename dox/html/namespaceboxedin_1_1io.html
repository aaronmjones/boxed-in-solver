Level File

Level File Characters

A level file is composed of the following characters 

 ' space
 x wall
   floor (space character)
 + box
 * gear
 r red switch
 g green switch
 b blue switch
 y yellow switch
 R red gate
 G green gate
 B blue gate
 Y yellow gate
 @ exit

Level File Example

Game 1, Level 37 (levels/01/37.txt): 

''''''''''
xxxxxxxxxx
xp   +   x
xr Y + x x
x  x*x   x
x   +xx yx
xxx++ x x'
x +     x'
'x xx x x'
'x      x'
'xRxxxxGx'
'xg*x'x@x'
'xxxx'xxx'
''''''''''
