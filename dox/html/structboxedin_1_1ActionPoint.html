Detailed Description

This is used to generate successors to a BoxedInNode.

The algorithm expands each BoxedInNode to generate a list of BoxedInNode's that can be reached by doing some action. Possible actions are: moving a box, stepping on a gear or stepping on the exit. An ActionPoint's coord is one of:

  • coordinate where the player can push an adjacent box,
  • coordinate of a gear
  • coordinate of the exit

An ActoinPoint's path is the path from the BoxedInNode's player coordinate to the ActionPoint coordinate and is used to create the successor BoxedInNode's path.
