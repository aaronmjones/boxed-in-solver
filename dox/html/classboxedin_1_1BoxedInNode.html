Detailed Description

This is the search node for A* open set and closed set.

Each BoxedInNode is uniquely identified by key, which is a hash of the character map representation of the state of the node.

Certain data fields are only needed while a BoxedInNode is in the open set. These fields have been aggregated into the OpenSetData struct and are deleted once the node is moved into the closed set in order to reduce the amount memory used.
